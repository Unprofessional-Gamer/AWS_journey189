# AWS Automation Project

This project contains Python scripts for AWS automation tasks using boto3.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Configure AWS credentials:
   - Create a `config/.env` file with your AWS credentials
   - Or configure AWS CLI using `aws configure`

## Project Structure

- `src/`: Contains main AWS automation scripts
- `tests/`: Unit tests
- `config/`: Configuration files and credentials
- `requirements.txt`: Project dependencies

## Getting Started

1. Install the required dependencies
2. Configure your AWS credentials
3. Run scripts from the `src` directory
